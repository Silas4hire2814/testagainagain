How does a search engine work?
First, search engine bots or spiders crawls through the internet downloading web pages and following links to those websites that are available. Next a collection of websites that the web spiders collected is put in an algorithm that matches the search term which should produce the best results.
Why do different search-engines give different ranking to websites for the same search phrase?
Unless someone was somehow cheating or stealing, most search bot spiders are made differently. They have different methods of getting web pages and search results. A web crawler from yahoo will diffrent and get different results from the same search compared to a search from google or bing
What is a <meta /> tag?
“The tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable. Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.”
What is the purpose for Keyword and Description meta tags?
I would have to think that keyword and description meta tags help search bots find certain information on your web page and tell it what the website is about.
Why might you include meta tags in your pages?
That way people can find my website easier when looking for certain information or when they are looking for something. My page has a higher chance of showing up.
