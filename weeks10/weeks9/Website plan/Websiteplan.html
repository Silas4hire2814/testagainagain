Project plan

title

shrine of resurrection

Goal

A website dedicated for Link mains to find other Link mains and up their game or learn more about the character in Super Smash Brothers Ultimate.

Target Audience

For Link mains looking for more information, new link players looking for more information, or just Super Smash Bros players.

Main Features

A frame data section, Link main discord server, a link to the smash boards server and guides to getting better.

Personality

The site should have an easy going layout but should kind of have a simple royal layouy with the navy blue and a dark red. The site should look not too professional but have a mood that is dedicated to informing people on the character.
